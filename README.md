Smallisp
=====

Lisp Implementation in C++.


## Intro

Simple Lisp implementation in C++.


## Dependencies
### Building 
 * clang++
 * C++ 14
 * libedit
 * googletest (for testing)


## Usage
run `$ make all`. This should generate a `repl` executable.

Execute `$ ./repl`.
